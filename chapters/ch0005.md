#### FLAGs

时间过去了一个月，或许是上一批海盗死的太惨，海盗们都没有再来抢劫<ruby>新阿拉斯加<rt>NEW ALASKA</rt></ruby>州。**UNE DES MOINES**的专属船坞已经建成，Y称它为母港。于是，**UNE DES MOINES**就被停放在母港中，而五个人开着五架小型的船只出去旅游。正当他们游览到一半时，收到来自母舰的消息：

```
from: 哨兵阵列
边境存在多支不明外形舰队，暂命名为Gamma Aliens。请前往处理。
```

以及

```
from: Kizuna AI
爱酱的温馨提醒！鉴于Gamma Aliens入侵再先，我们可以稍微放开手脚哦！
```

“爱酱真是可爱啊……”Mr. Y在脑海中开始发电。不过，他很快就恢复过来，向其余FLAGs成员下达命令。

“跃迁至<ruby>新巴罗<rt>NEW BARROW</rt></ruby>“

随后，Mr. Y小声发表了一句感慨。

“FLAG的WS形态真好用啊……”

---

<ruby>新巴罗<rt>NEW BARROW</rt></ruby>，恒星系外围。

*Gamma Aliens*的战舰陈列于<ruby>新巴罗<rt>NEW BARROW</rt></ruby>的第三颗行星旁。<ruby>新巴罗<rt>NEW BARROW</rt></ruby>-3因为其大量的稀有金属矿藏而出名，尤其是计划总常温超导体中的成分。虽然地球联合国更常使用超空间进行远距离输电，但是在更普及的短途场景下，超导输电仍然是最常用的。鉴于此，<ruby>新巴罗<rt>NEW BARROW</rt></ruby>成为了一个重要的矿业基地。然而，出于边境州糟糕的管理，这里的换防存在很大的漏洞。在某次换防的间隙，*Gamma Aliens*长驱直入，控制了这个星系，而国民警卫队根本就没有发现。

```
from: Kizuna AI
根据哨兵阵列的数据，大概有十条左右的主力舰和更多的小型舰船哦！是否需要爱酱提供远程支援呢？
```

Mr. Y迟疑了一下，向Kizuna AI索要了快子光矛和聚能电弧，配发给FLAG 2和FLAG 3。很快，随着空间的一阵微乎其微的波动，两台外挂装备便被动跃迁到了他们附近。

“完全歼灭？还是留一些活口？”

Kizuna AI的声音直接在Y的耳边响起。“留活口可能会造成不必要的麻烦哦……”

于是，FLAG 1蠢蠢欲动。伴随着驾驶员1中二的招式介绍，“FLAG 1，切换形态！”，FLAG 1的舰体结构开始发生变化，一台漆黑的MS出现在宇宙空间中。

“啊哈哈哈，这就是联合国的最新科技！外星杂碎受死吧……”。Y打断了1的神经病发言，“这里离敌军大概有几十个天文单位，你打算以MS形态飞过去吗？”

通讯中一片寂静。驾驶员1默默地把FLAG 1切换回WS形态。由于刚才毫无遮蔽的变形，*Gamma Aliens*已经发现了他们。攻击很快抵达，只不过瞄准的似乎是FLAGs几秒前的位置。

“来而不往，非礼也。如果我们不回击，那不就是非礼……”。驾驶员2和3各自接通了快子光矛和聚能电弧，向敌方舰队回敬了一片弹幕。

“*Gamma Aliens*的防御水平还可以，节能模式下的快子光矛需要两次点射才能击穿主力舰的装甲带……”。驾驶员2在攻击的同时顺便向Kizuna AI发送了分析数据，虽然这些事情可以被Kizuna AI全自动完成，但是给人类一些仪式感和参与感总是好的。

火球不断的在*Gamma Aliens*的舰队中腾起。FLAGs也正在不断地接近。这样的战斗显得十分的无聊，于是，FLAG 3进行了一个剧烈的机动，脱离了航线，随后使聚能电弧完全充能，打出了一个范围攻击，在敌军舰队中扫出一个空缺。与此同时，FLAG X在高速前行中瞬间完成了型态切换，抽出了两把光剑，一个短距离跃迁机动进入敌群。看到这里，其余几台FLAG不约而同地停了下来。

“队长又要装逼了。”

“……”

“不留活口的话装逼有什么用？？？”

FLAG X的驾驶舱中，Y极度中二的喊出招式，“最 大 展 开 ！”，FLAG X手中的光剑合二为一，功率爆发，展开为一把长度惊人的巨剑。FLAG X手握光剑作出360度转体机动，战斗结束。
